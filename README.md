# Simplicity in Action

A simple and clean login interface project built using HTML5.  

---

## Features

- Login modal using HTML `<dialog>`
- Expandable form section using `<details>`
- User input fields (username, password)
- Course selection dropdown
- Gender selection using radio buttons
- Clean and minimal UI structure
